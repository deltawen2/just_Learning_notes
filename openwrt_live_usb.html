openwrt 系統製作安裝

  1. 到openwrt庫下載對應型號

這裡用x86_x64版本

https://firmware-selector.openwrt.org/

  1. 整合首次啟動時執行的指令碼 (uci-defaults)進img

在首次啟動時執行的指令碼欄位輸入區網自訂IP

這裡以192.168.99.5/24為例

#設定lan IP
uci set network.lan.ipaddr='192.168.99.5'

#將設定永久寫進文件
uci commit network

#重啟網路服務
/etc/init.d/network restart
  1. 點選要求建製紐後下載img
  2. 燒錄img進usb
sudo dd if=openwrt-24.10.2-30ec32c3854d-x86-64-generic-ext4-combined-efi.img of=/dev/sdx bs=4M status=progress oflag=sync



＃sudo: 以管理者權限執行該命令。
＃dd: 用於複製和轉換檔案的命令行工具。
＃if=openwrt-24.10.2-30ec32c3854d-x86-64-generic-ext4-combined-efi.img: 指定輸入檔案，這裡是您的 OpenWrt 映像檔。
＃of=/dev/sdx: 指定輸出檔案，請將 /dev/sdx 替換為您實際的目標裝置名稱！
＃bs=4M: 設定區塊大小為 4MB，這通常是一個較好的傳輸效率設定。
＃status=progress: 在複製過程中顯示進度。
＃oflag=sync: 強制將所有寫入操作同步到磁碟，這可以提高資料寫入的可靠性，但可能會稍微降低速度。

系統安裝完成

  • 插入電腦開機
  • 從同區網電腦網頁輸入192.168.99.5即可進入設定界面
