如何在 Ubuntu 24.04 中新增一般使用者 omagawen 並加入 sudo 群組

這是一個詳細的步驟指南
Ubuntu 24.04 系統中新增一個名為 omagawen 的使用者，並賦予他執行管理員指令（sudo）的權限。

步驟一：開啟終端機

步驟二：新增使用者帳號 omagawen

  • 使用 adduser 指令來新增使用者。這個指令會引導你設定新使用者的密碼和一些基本資訊。
sudo adduser omagawen

執行此指令後，系統會要求你輸入當前擁有 sudo 權限的使用者（例如你的 deltawen 帳號）的密碼。輸入密碼後，請按照提示完成以下步驟：

  1. 設定新密碼： 系統會提示你為 omagawen 設定新密碼。請輸入一個至少 8 個字元長度的強密碼，然後再次輸入以確認。

    New password:
Retype new password:
  • 如果密碼太短，系統會提示 BAD PASSWORD: The password is shorter than 8 characters，請重新輸入符合要求的密碼

  1. 輸入使用者資訊（選填）： 系統會詢問一些關於 omagawen 的個人資訊，例如全名、房間號碼、電話等。這些都是選填的，如果你不需要填寫，可以直接按 Enter 跳過。

    Changing the user information for omagawen
Enter the new value, or press ENTER for the default
    Full Name []:
    Room Number []:
    Work Phone []:
    Home Phone []:
    Other []:

  2. 確認資訊： 最後，系統會要求你確認所有輸入的資訊是否正確。輸入 y 然後按 Enter 即可確認。

    Is the information correct? [Y/n] y

完成這些步驟後，omagawen 這個使用者帳號就已經成功建立，並擁有自己的家目錄 (/home/omagawen)。

步驟三：將 omagawen 加入 sudo 群組

為了讓 omagawen 能夠執行 sudo 指令，你需要將他加入到 sudo 群組。請確保你當前是以擁有 sudo 權限的帳號（例如 deltawen）來執行這個指令。

Bash

sudo usermod -aG sudo omagawen

  • sudo: 以管理員權限執行指令。

  • usermod: 用於修改使用者帳號的指令。

  • -aG sudo: -a 表示追加（append），-G 表示指定群組。這會將 omagawen 加入到 sudo 群組，而不會移除他原有的群組。

  • omagawen: 你要加入群組的使用者名稱。

執行這個指令後，系統會要求你輸入當前使用者的密碼（例如 deltawen 的密碼）。

步驟四：驗證 omagawen 是否已成功加入 sudo 群組

你可以透過以下指令來確認 omagawen 是否已擁有 sudo 權限：

  1. 登出並重新登入 omagawen 帳號： 為了讓群組變更生效，你必須讓 omagawen 登出當前會話，然後再重新登入。

  2. 以 omagawen 身份執行 id 指令： 登入 omagawen 帳號後，打開終端機，執行：

    id omagawen

    在輸出結果中，你應該會看到 sudo 出現在 groups= 的列表中，例如：

    uid=1001(omagawen) gid=1001(omagawen) groups=1001(omagawen),4(adm),24(cdrom),27(sudo),30(dip),46(plugdev),120(lpadmin),131(lxd),132(sambashare)

    （注意 sudo 群組的存在）

  3. 以 omagawen 身份測試 sudo 指令：

    Bash

    sudo apt update

    如果一切設定正確，這次系統會要求你輸入 omagawen 的密碼，然後成功執行 apt update。

完成以上步驟後，你就在 Ubuntu 24.04 中成功新增了 omagawen 這個使用者，並賦予了他執行管理員指令的權限。
