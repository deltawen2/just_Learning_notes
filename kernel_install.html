🧰 安裝 XanMod RT 核心並設為預設開機選項（完整流程）

🟩 步驟一：下載核心套件

前往 XanMod 官方 SourceForge 頁面下載以下 .deb 套件：

🔗 XanMod RT 5.10.59-rt51-xanmod1

下載以下三個檔案（64-bit）：

  • linux-image-5.10.59-rt51-xanmod1_*.deb
  • linux-headers-5.10.59-rt51-xanmod1_*.deb
  • linux-libc-dev_*.deb（可選）

✅ 建議放在 ~/Downloads/xanmod 資料夾中集中管理

🟨 步驟二：安裝核心套件

cd ~/Downloads/xanmod
sudo dpkg -i *.deb
sudo apt --fix-broken install

若有依賴錯誤，--fix-broken 會自動補齊

🟦 步驟三：確認核心已安裝

ls /boot/vmlinuz-*

應看到：

/boot/vmlinuz-5.10.59-rt51-xanmod1

🟧 步驟四：設定 GRUB 預設開機核心

編輯 GRUB 設定檔：

sudo nano /etc/default/grub

修改以下行：

GRUB_DEFAULT="Advanced options for Ubuntu>Ubuntu, with Linux 5.10.59-rt51-xanmod1"
GRUB_TIMEOUT_STYLE=menu
GRUB_TIMEOUT=5

✅ 使用名稱方式比索引更穩定，避免 GRUB 更新後順序改變

🟥 步驟五：更新 GRUB 並重啟

sudo update-grub
sudo reboot

🟦 步驟六：確認是否成功載入 XanMod 核心

開機後執行：

uname -r

預期輸出：

5.10.59-rt51-xanmod1
